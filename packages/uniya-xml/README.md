# Uniya XML framework 

Reader and writer for XML nodes with some extensions.

See also: Uniya Set (https://www.npmjs.com/package/uniya)

## Contains

- [x] [Typescript](https://www.typescriptlang.org/) 3.x

### Build tools

- [x] [Webpack](https://webpack.github.io) 4.x

## Setup


```
$ npm install
```

## Testing

```
$ npm test
```

## Build

```
$ npm run build
```
